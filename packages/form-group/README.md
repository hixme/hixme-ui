# hixme-ui form-group

```bash
npm i --save @hixme-ui/form-group
```
[View it live](https://hixme.github.io/hixme-ui/form-group)

## Usage

```javascript
import FormGroup from '@hixme-ui/form-group'

<FormGroup>
  <Label>Name</Label>
  <Input name='name' />
  <Text error>Name not valid</Text>
</FormGroup>
```
