# hixme-ui checkbox

```bash
npm i --save @hixme-ui/checkbox
```
[View it live](https://hixme.github.io/hixme-ui/checkbox)

## Usage

```javascript
import Checkbox from '@hixme-ui/checkbox'

<Checkbox text='Checkbox Label' />
```

## Props

| Name            | Type        | Default        |
| --------------- | ----------- | -------------- |
| text            | string      | N/A            |
| small           | boolean     | false          | 
| disabled        | boolean     | false          |
