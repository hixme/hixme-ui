# hixme-ui title

```bash
npm i --save @hixme-ui/title
```

Title extends the @hixme/text component properties


## Usage

```javascript
import Title from '@hixme-ui/title'


<Title>Big loud text!</Title>

<Title jumbo>Jumbo title for attention</Title>

```

## Props

| Name            | Type        |
|-----------------|-------------|
| jumbo           | bool        |
| small           | bool        |
| smaller         | bool        |
| smallest        | bool        |
