# hixme-ui label


```bash
npm i --save @hixme-ui/label
```

[View it live](https://hixme.github.io/hixme-ui/label)

## Basic Usage

Use it anywhere you would use a form label.

```javascript
import Label from '@hixme-ui/label'

<form>
  <Label>First name</Label>
  <input name='firstname' required />
</form>
```
