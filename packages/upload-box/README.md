# hixme-ui upload-box

```bash
npm i --save @hixme-ui/upload-box
```
[View it live](https://hixme.github.io/hixme-ui/upload-box)

## Usage

```javascript
import UploadBox from '@hixme-ui/upload-box'

<UploadBox
  compact
  onDrop={doSomething}
/>
```

## Default Props

| Name          | Type      | Default  |
| ------------- | --------- | -------- |
| compact       | boolean   | false    |
| value         | string    | ''       |
| onDrop        | function  | N/A      |
