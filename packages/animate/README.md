# hixme-ui animate

```bash
npm i --save @hixme-ui/animate
```

## Usage

```javascript
import Animate from '@hixme-ui/animate'

<Animate bounce time='1s' />

```

## Props
Add any animation from [https://daneden.github.io/animate.css/](Animate.css) as a boolean prop.

| Name            | Type        |
---------------------------------
| time            | string      |
| delay           | string      |
| count           | string      |
| trigger         | string      |
| display         | string      |
