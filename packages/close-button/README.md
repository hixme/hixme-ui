# Close Button

```bash
npm i --save @hixme-ui/close-button
```
[View it live](https://hixme.github.io/hixme-ui/close-button)

## Usage

```javascript
import CloseButton from '@hixme-ui/close-button'

<CloseButton onClick={onClick} />
```

## Props

| Name            | Type        | Default        |
|-----------------|-------------|----------------|
| iconSize        | string      | '1.8'          |
| onClick         | string      | () => null     |
| label           | string      | 'CLOSE'        |
| noLabel         | string      | false          |
