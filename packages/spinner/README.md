# Spinner

```bash
npm i --save @hixme-ui/spinner
```
[View it live](https://hixme.github.io/hixme-ui/spinner)

## Usage
Wrap with things you want to spin!

```javascript
import Spinner from '@hixme-ui/spinner'

<Spinner animate>
  <Icon cog />
</Spinner>
```

## Props

| Name            | Type        | Default        |
|-----------------|-------------|----------------|
| animate         | boolean     | false          |
| duration        | string      | '1s'           |
| ease            | boolean     | false          |
| linear          | boolean     | true           |
