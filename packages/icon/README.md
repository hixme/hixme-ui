# Icon

```bash
npm i --save @hixme-ui/icon
```
[View it live](https://hixme.github.io/hixme-ui/icon)

## Usage

```javascript
import Icon from '@hixme-ui/icon'

<Icon menu />
```

## Props

| Name            | Type        | Default        |
| --------------- | ----------- | -------------- |
| color           | string      | black          |
| size            | number      | 1              |
| fontSize        | number      | 14             |
