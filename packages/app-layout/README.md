# hixme-ui app-layout

```bash
npm i --save @hixme-ui/app-layout
```
[View it live](https://hixme.github.io/hixme-ui/app-layout)

AppLayout is an all-in-one menu and sidebar.

## Usage

```javascript
import AppLayout from '@hixme-ui/app-layout'

<AppLayout>
  {children}
</AppLayout>
```

## Props

| Name            | Type        | Default        |
| --------------- | ----------- | -------------- |
| sidebarWidth    | string      | '225px'        |
