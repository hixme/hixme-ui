# hixme-ui print

```bash
npm i --save @hixme-ui/printc
```
[View it live](https://hixme.github.io/hixme-ui/print)

## Usage

```javascript
import Print from '@hixme-ui/print'

<Print />
```

## Props

| Name            | Type        | Default        |
|-----------------|-------------|----------------|
| label           | string      | Print          |
| showIcon        | bool        | true           |
| showLabel       | bool        | true           |
