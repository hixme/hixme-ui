# hixme-ui text

```bash
npm i --save @hixme-ui/text
```

## Usage

```javascript
import Text from '@hixme-ui/text'

// text success
<Text success>I said good day sir!</Text>

// paragraph warning
<Text p warning>Are you sure?</Text>
```
